# 炙热竞技

  应用介绍: "炙热竞技"app是一款购买竞技手办(王者手办、阴阳师手办、绝地求生手办)的应用。

  应用功能: 
  
    手办分类：王者荣耀手办、阴阳师手办、绝地求生手办等类型竞技手办。
    下单方式：直接下单结算、或者加入购物车批量结算(建议此法)。
    购买流程：线上下单，货到付款，商家免费配送。
    积分商城：订单完成后获得对应积分数量，积分可直接在积分商城进行兑换商品。
    订单：商品下单的订单和积分兑换的商品，均在订单栏目中。
    设置：通用设置和收货地址管理等。
